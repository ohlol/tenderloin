Use Tenderloin

to collect and display data about system metrics. You can view or process the data with any standard HTTP client or library (curl, urllib2, Net::HTTP, etc.), receive alerts, and graph timeseries-based data. It's a snap to set up and you can write plugins in the language of your choosing.
